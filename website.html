Welcome to the VISHLESHNAM project website. This project focuses on developing video analytics solutions for crowd behaviour analysis using multiple sensors.

About

Project Overview

Project Investigator: Dr. Debi Prosad Dogra, Associate Professor, Computer Science & Engineering, IIT Bhubaneswar

Duration: February 2024 to February 2027

Budget: Rs. 44.42 Lakhs

Funding Agency: IRDE Dehradun, DRDO

Sanction No.: ECS/IRDE/23-24/008 (CARS)

Objectives

  • AI Algorithm Development: Detect suspicious activities of intruders targeting women in various environments.
  • Automatic Alarm Generation: AI-driven methodologies for alarm generation to assist women in distress.
  • Vehicle Surveillance: Detect non-standard or improperly installed number plates within a restricted zone.
  • Crowd Analysis: Develop deep learning models to count males and females in crowds and analyze activities.
